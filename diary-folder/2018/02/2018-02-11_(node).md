一开始可能会觉得迭代器很麻烦, 其实可以封装一个简单的函数来重复利用它:
```
function step(gen){
  const g = gen();
  let last;

  return () => {
    last = g.next(last).value;
    return last;
  }
}

function *foo() {
  yield 1;
  yield 2;
  yield 3;
}

const f = step(foo);
console.log(f()); // 1
console.log(f()); // 2
console.log(f()); // 3
```
